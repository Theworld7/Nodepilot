## v{version}

### 变更

- **日志窗口增强**：支持 ANSI 彩色渲染输出，新增行选择模式（点击行号或「选择」按钮进入），支持 Shift 范围多选、全选/取消全选，一键复制选中行并自动退出选择模式
- **项目配置优化**：默认执行命令改用下拉选择器（Select），选项面板宽度受视口约束，同时展示脚本名与命令值
- **Dev Server 稳定性修复**：修复 macOS 上 script PTY 包装导致子进程 stdin 关闭、Vite 启动后立即退出的问题，通过管道保持 stdin 打开
- **UI 细节**：版本面板头部增加柔和阴影，提升立体感

### 安装

下载 `nodepilot_{version}_aarch64.dmg`，双击挂载后将 `nodepilot.app` 拖入 Applications 文件夹。

macOS 首次启动时，若提示「无法验证开发者」，请前往 **系统设置 → 隐私与安全性** 中点击「仍要打开」。
